# Ross Farhi Portfolio
## Creator
- Ross Farhi
## Description
- This project is a redesign of my current portfolio that I use to showcase my graphic design work throughout my professional career. The current website that I use is fine, but feels boring and I want to try to make it a little more interesting, while still keeping the main focus on the graphics themselves. There are going to be a few different main groups of projects: UC Lacrosse graphics, UC Baseball graphics, UC graphics as a whole, Miscellaneous graphics which are ones I've made for fun, and replica NCAA College Football Video Game covers showcasing various players.
## Target Audience
- My **primary** audience is going to be potential employers looking at my portfolio who will hopefully offer me a job in the near future. My **secondary** audience will be other college students or anyone just getting into graphic design who might want to look at my portfolio for either ideas, inspiration, or to compare their own work.
## Goals
- The project goals for this are to: showcase my graphics to employers, keep the focus on the graphics and not the website, and to make a working and interactive website that is functional.
